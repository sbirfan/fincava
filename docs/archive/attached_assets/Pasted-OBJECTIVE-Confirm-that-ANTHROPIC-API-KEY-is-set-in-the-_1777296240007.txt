IMPLEMENTATION
## OBJECTIVE
Verify ANTHROPIC_API_KEY is active and the scoring pipeline can reach Anthropic AI.

## PREFLIGHT
1. Open Replit → Secrets panel
2. Confirm ANTHROPIC_API_KEY is listed (value hidden is OK)

## STEPS
1. In Replit shell, run:
   curl -X POST http://localhost:5000/api/suppliers/onboard \
     -H "Content-Type: application/json" \
     -d '{"test": true}'
   (This will fail with 400/validation error — that's fine; confirms server is running)

2. Check the Replit Secrets panel: ANTHROPIC_API_KEY must be present.

3. POST a minimal onboarding request with a real payload:
   curl -X POST http://localhost:5000/api/suppliers/onboard \
     -H "Content-Type: application/json" \
     -H "Authorization: Bearer <valid-token>" \
     -d '{"farmName": "Test Farm", "location": "Santander", ...}'
   
   Wait 5s, then:
   SELECT * FROM ai_outputs ORDER BY created_at DESC LIMIT 1;
   
   Expected: row with valid score_data JSON, NOT NULL, NOT error.

4. If scoring returns 500 with "API key" error:
   → Update ANTHROPIC_API_KEY in Replit Secrets with valid key
   → Restart the Replit service
   → Repeat step 3
