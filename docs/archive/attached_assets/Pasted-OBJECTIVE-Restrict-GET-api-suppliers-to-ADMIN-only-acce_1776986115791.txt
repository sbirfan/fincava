THE FIX
Two targeted changes — no redesign.

Change 1 — Add authentication guard
Apply requireAuth middleware to GET /api/suppliers
Result: no token → 401

Change 2 — Restrict to ADMIN role
Apply requireAdmin middleware to GET /api/suppliers
Result: non-admin authenticated user → 403

REQUIRED ROUTE SHAPE
Before: router.get('/api/suppliers', handler)
After:  router.get('/api/suppliers', requireAuth, requireAdmin, handler)

Middleware order is MANDATORY: requireAuth first, requireAdmin second.
Do not reverse. Do not combine into a single middleware.

IMPORTANT DESIGN DECISION
GET /api/suppliers becomes ADMIN-only.
Full response payload remains unchanged — no field sanitization.
The buyer-safe surface is GET /suppliers/marketplace (public, unchanged).
